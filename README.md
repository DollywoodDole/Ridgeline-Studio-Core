# Ridgeline Studio

**We upgrade local businesses into digital assets.**

---

## About

Ridgeline Studio is a boutique web agency based in Saskatchewan. We specialize in converting outdated or underperforming local business websites into conversion-ready digital assets — built on modern stacks, deployed fast, and designed to generate real leads.

Solo-operated by Dalton. Every client gets a dedicated repo, a clean codebase, and a site that actually works.

## Stack Standards

All Ridgeline Studio projects are built on:

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
- **CMS (when needed):** TBD per project

Repo types:
- `static-upgrade` — brochure/informational sites
- `cms-upgrade` — content-managed sites
- `ecomm-upgrade` — e-commerce sites
- `lead-gen-upgrade` — conversion-focused lead generation sites

## Onboarding SOP

New client projects follow a 6-step process. See the full SOP at [docs/onboarding.md](./docs/onboarding.md).

1. Create GitHub repo
2. Clone locally
3. Scaffold (`CLAUDE.md`, `CLIENT.md`, `.env.example`)
4. Deploy to Vercel
5. Open VSCode workspace
6. Start Claude Code session
